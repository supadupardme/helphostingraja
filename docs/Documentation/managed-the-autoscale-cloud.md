<div style="text-align: left;"><span style="font-size: 14pt;"><br/><strong>Managed Autoscale Cloud<br/><br/></strong></span>In cloud computing, scaling is a process by which we can add or remove storage and network services to meet the demands a workload makes for resources so as to maintain the availability and the performance when the utilization increases. Scaling mainly refers as adding or reducing the number of servers i.e instances which are active. being leveraged from the workloads resource demands. Scaling up and scaling down can be refered to as two dimensions across the resources and hence therefore capacity can added.</div>
