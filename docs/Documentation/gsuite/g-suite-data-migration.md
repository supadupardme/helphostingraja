<h1><b>G Suite - Data Migration from cPanel/Plesk or anyother to G Suite</b></h1>
<br/>
<ol>
<li><span style="font-weight: 400;"> Login with your admin login details in</span><a href="https://admin.google.com"><span style="font-weight: 400;"> https://admin.google.com</span></a></li>
</ol>
<br/>
<ol start="2">
<li><span style="font-weight: 400;"> After logging in, please click “Data Migration”. See the below image.</span></li>
</ol>
<br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/data-migration-icon.jpg" alt="Admin console " width="238" height="686" /> <br/><br/>
<ol start="3">
<li><span style="font-weight: 400;"> Then choose the migration option, the one you want to migrate.</span></li>
</ol>
<br/><br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/choose-migration-option.jpg" alt="one you want to migrate" width="413" height="373" /> <br/><br/>
<ol start="4">
<li><span style="font-weight: 400;"> Then click on continue. You will get the screen as mentioned below.</span></li>
</ol>
<br/><br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/add-connection-protocols.jpg" alt="add-connection-protocols" width="409" height="535" /> <br/>
<p><span style="font-weight: 400;">In this on </span><b>“Migration source”</b><span style="font-weight: 400;">, you can choose any one option from the below.</span></p>
<br/>
<ul>
<li style="font-weight: 400;"><span style="font-weight: 400;">G Suite</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">Gmail</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">GoDaddy</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">Microsoft Exchange 2003 / 2007 / 2010 / 2013,2016 </span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">Microsoft Office 365</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">I don’t know /other IMAP servers.</span></li>
</ul>
<br/><br/><br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">Choose the option IMAP server, if you are using cPanel server.</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">In “Connection protocol”. Choose IMAP if migrating from cPanel</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">And then enter IMAP server Name and Port. Port number should be 993.</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">You have to enter the hostname of your server, which is pointing your domain.</span></li>
<li style="font-weight: 400;"><span style="font-weight: 400;">And the hostname should have SSL. </span></li>
</ol>
<br/><br/>
<p><b>Note: If SSL is not installed, you need to contact your hosting provider.</b></p>
<br/>
<ol start="6">
<li><span style="font-weight: 400;">6</span><b>. </b><span style="font-weight: 400;">In the next “</span><b>Role account</b><span style="font-weight: 400;">”, you have to enter anyone email id and password which is already you using with service provider.</span></li>
</ol>
<br/>
<ol start="7">
<li><span style="font-weight: 400;"> After entering all the details, please click on</span><b> “CONNECT”</b><span style="font-weight: 400;">. Then it will connect and complete the authentication.</span></li>
</ol>
<br/>
<ol start="8">
<li><span style="font-weight: 400;"> If the connection fails, please check the hostname or IP address. Please enter the correct hostname / IP address and reconnect.</span></li>
</ol>
<br/>
<ol start="9">
<li><span style="font-weight: 400;"> If it's not connecting again contact your <a href="https://www.hostingraja.in/" target="_blank" rel="noopener noreferrer">hosting provider</a> and get the correct hostname / IP address and connect.</span></li>
</ol>
<br/><br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/get-connect-after-entered.jpg" alt="get-connect-after-entered" width="400" height="481" /> <br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">After once it successfully connected, you will be redirected to the migration type dashboard.</span></li>
</ol>
<br/><br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/choose-option-to-migrate-details.jpg" alt="choose-option-to-migrate-details" width="378" height="365" /> <br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">Here you have to choose from which date you have to migrate the data and choose options to email deleted/junk/or exclude folders.</span></li>
</ol>
<br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">And click </span><b>“SELECT USERS”</b><span style="font-weight: 400;">.</span></li>
</ol>
<br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">Finally, you will find the dashboard like below. Here you need to enter email id and password of existing cPanel / or you can upload bulk email ids as a file in the below format which is shown in the image.</span></li>
</ol>
<br/> <br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/add-migration-email-id.jpg" alt="add-migration-email-id" width="395" height="235" /> <br/>
<p><b>Note: You have to create all the email ids in G Suite whichever you want to migrate, and then it will perform.</b></p>
<br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">You can migrate email ids in two ways. By clicking the yellow button shown in the screenshot.</span></li>
</ol>
<br/>
<p><b>For Single Email Id / One by one:</b></p>
<br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">First, you have to to choose </span><b>“select user”</b><span style="font-weight: 400;"> by hovering the yellow plus icon.</span></li>
</ol>
<br/>
<ol>
<li style="font-weight: 400;"><span style="font-weight: 400;">You have entered the email id and password of which one you want to migrate from and type the email id to which you want to migrate.</span></li>
</ol>
<br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/migration-dashboard.jpg" alt="migration-dashboard" width="340" height="673" /> <br/>
<p><b>Note: Before entering the migrate to email you have to create the user (Email Id) in your admin dashboard. Then you have to choose the email id by typing in migrate to.</b></p>
<ol start="3">
<li><span style="font-weight: 400;"> Then click on </span><b>“START”.  I</b><span style="font-weight: 400;">t will initialize the migration. Once the migration is done, it will show as completed.</span></li>
</ol>
<br/>
<ol start="4">
<li><b> Bulk email id migration:</b></li>
</ol>
<br/>
<ol start="5">
<li><span style="font-weight: 400;"> You have to choose “</span><b>Select multiple users</b><span style="font-weight: 400;">” by hovering the yellow plus icon.</span></li>
</ol>
<br/><br/> <img src="https://image.hostingraja.in/images/gsuite/tutorials/bulk-email-migration.jpg" alt="bulk-email-migration" width="406" height="400" /> <br/>
<ol start="2">
<li><span style="font-weight: 400;"> Create a text file as in the given format and upload it to start your bulk email migration.</span></li>
</ol>
