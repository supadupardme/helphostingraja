<h1 dir="ltr" style="text-align: center;"><strong><span style="font-size: x-large;">How to Open Port in <a href="https://www.hostingraja.in/server/vps-servers//windows-vps-hosting/">Windows Virtual Server</a></span></strong></h1>
<h1 dir="ltr">What is Port?</h1>
<p dir="ltr">Port numbers are associated with TCP/IP network connections. A port number is a way to identify a specific process to which an Internet or other network message is to be forwarded when it arrives at a server. They allow different applications on the same computer to share network resources simultaneously. Home network routers and computer software work with these ports and sometimes support configuring port number settings.</p>
<h1 dir="ltr">How Does Port Numbers Work?</h1>
<p dir="ltr">Port numbers relate to network addressing. In TCP/IP networking, both TCP and UDP utilize their own set of ports that work together with IP addresses.</p>
<p>In both TCP and UDP, port numbers start at 0 and go up to 65535. Numbers in the lower ranges are dedicated to common Internet protocols (like port 80 for HTTP and port 443 for HTTPS).</p>
<p dir="ltr">This article will be showing you how to open ports in your Linux servers firewall. This can be very useful, and also dangerous, so be sure you know what port you want to open up. Hackers can use these open ports to gain unauthorized access to your server, so be careful!</p>
<h1 dir="ltr">To open a port in Windows Firewall</h1>
<p dir="ltr">Open Start<img src="https://lh5.googleusercontent.com/IuYgHV4lY9NXU7Ryn9NNqtcYN4LViEAcb0JYZQ1OsDxbjKv1HRBNMpgk2hLuiLweWosW8QabQIwYAXXa8CTiyqoXcJsQeOlqEYlrcmu8IpqjCvyifOEL4-W4GzJvoVAdjM-yKjpe" alt="Win_0.png" width="30" height="30" border="0" /> . Click the <a href="https://www.hostingraja.in/">Windows </a>logo in the bottom-left corner of the screen.</p>
<div dir="ltr"> </div>
<p dir="ltr"><img src="https://image.hostingraja.in/images/how-to-open-port-in-windows.png" alt="Win_2.png" width="100%" border="0" /></p>
<p>In Windows Firewall, click the Advanced Settings.</p>
<p dir="ltr"><img src="https://image.hostingraja.in/images/how-to-open-port-in-windows1.png" alt="Win_3.png" width="100%" border="0" /></p>
<p>Click 'Inbound Rule' and Select 'New Rule' <br/><br/><img style="font-family: Tahoma, Helvetica, Arial, sans-serif; font-size: 12.16px;" src="https://image.hostingraja.in/images/how-to-open-port-in-windows2.png" alt="Win_4.png" width="100%" border="0" /> <br/><br/>Check the "Port" option, then click Next. This will allow you to manually choose the ports you want to open. <img style="font-family: Tahoma, Helvetica, Arial, sans-serif; font-size: 12.16px;" src="https://image.hostingraja.in/images/how-to-open-port-in-windows3.png" alt="Win_5.png" width="100%" border="0" /><br/><br/>In the Specific Local Ports dialog box type the port number and click next Select Control Panel</p>
<p dir="ltr"><img src="https://image.hostingraja.in/images/how-to-open-port-in-windows4.png" alt="Win_1.png" width="100%" border="0" /></p>
<p dir="ltr">Select Windows Firewall</p>
<p dir="ltr"><img src="https://image.hostingraja.in/images/how-to-open-port-in-windows5.png" alt="WIN_7.png" width="100%" border="0" /></p>
<p>Select the checkbox 'Allow the Connection' and click next. <img style="font-family: Tahoma, Helvetica, Arial, sans-serif; font-size: 12.16px;" src="https://image.hostingraja.in/images/how-to-open-port-in-windows6.png" alt="WIN_8.png" width="100%" border="0" /> <br/><br/>Select the checkbox of all three Domain, Private and Public and click next. <img style="font-family: Tahoma, Helvetica, Arial, sans-serif; font-size: 12.16px;" src="https://image.hostingraja.in/images/how-to-open-port-in-windows7.png" alt="WIN_9.png" width="100%" border="0" /><br/><br/>Specify the Firewall Rule name in the 'Name' dialog box and click 'Finish' to complete the 'Inbound' Firewall Rule.</p>
<p dir="ltr"><img src="https://image.hostingraja.in/images/how-to-open-port-in-windows8.png" alt="WIN_10.png" width="100%" border="0" /></p>
<p>Note: Follow the same steps to enable ‘Outbound’ Rule.<br/><br/></p>
